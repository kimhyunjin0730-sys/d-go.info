```
D-GO — 차세대 개인 보안 데이터 금고
```

---

### 제품 정의
```
D-GO(디고)는 '데이터 금고(Data Go-vault)'의 약자로,
외부 서버에 의존하지 않고 물리적으로 격리된 환경에서
디지털 자산과 아날로그 자산을 함께 보관하는
자기주권형(Self-Sovereign) 데이터 관리 솔루션입니다.
```

---

### 기술 스펙 테이블

| 항목 | 사양 |
|---|---|
| 인증 방식 | PUF(Physical Unclonable Function) 칩 기반 |
| 암호화 | AES-256 이중 암호화 |
| 무결성 검증 | SHA-256 해시 + TSA 타임스탬프 |
| 네트워크 | 폐쇄형 (통신 모듈 없음) |
| 데이터 입력 | 전용 Scanner / Voice Recording Device / 보안 USB |
| 데이터 출력 | 전용 보안 USB |
| 앱 플랫폼 | Android (전용 앱) |
| 파일 뷰어 | 한컴오피스 Viewer 연동 |
| 추가 보안 | 비인가 장치 즉시 차단 / 탈취 시 보안키 자동 소멸 |
| 아날로그 보관 | 인감·OTP·카드·통장 등 실물 자산 수납 공간 내장 |

---

### 저장 가능한 데이터 유형

1. **스캔본** — 계약서, 증명서, 보안 문서 (전용 Scanner 활용)
2. **음성본** — 회의록, 증거 녹취, 아이디어 메모 (STT 변환)
3. **파일본** — 법인 업무 자료, 개인 기밀 파일 (보안 USB)
4. **멀티미디어** — 고유 자산 가치가 있는 사진·영상

---

---

### D-GO App 설치 안내

> **Android 전용 앱입니다.**

#### 설치 순서

```
Step 1. 아래 [APK 다운로드] 버튼을 눌러 설치 파일을 내려받습니다.
Step 2. 기기 설정 → '출처를 알 수 없는 앱 설치' 허용 후 APK 실행.
Step 3. 설치 완료 후 앱 실행 → 권한 설정 화면에서 모든 항목 [허용].
Step 4. 플레이스토어에서 '한컴오피스 Viewer' 추가 설치 (파일 뷰어 필수).
```

**[APK 다운로드 버튼]** ← 실제 파일 링크 삽입

---

#### 초기 설정 3단계

```
① Dock 백업 저장 폴더 지정
   설정 화면 → [저장 경로 설정] → Vault 도킹 폴더 선택

② D-GO 자동복사 시간 간격 설정
   설정 화면 → [자동 동기화] → 원하는 주기 선택 (예: 1시간)

③ 보안키 생성 (AES-256)
   설정 화면 → [보안키 관리] → [새로 생성] 버튼 클릭
   ⚠️ 생성된 키는 반드시 별도 안전한 곳에 백업하세요.
```

---

### 도킹 연결 방법

```
1. D-GO Vault와 보안 태블릿을 도킹 스테이션에 물리적으로 결합합니다.
2. PUF 칩 인증이 자동으로 시작됩니다 (LED 점등 확인).
3. 생체 인증(지문 또는 PIN) 수행 후 암호화 채널 활성화.
4. 앱의 'TABLET DOCK(Vault)' 메뉴에서 연결 상태를 확인합니다.
```

### FAQ (자주 묻는 질문)

**Q. 한컴오피스 Viewer는 필수인가요?**  
A. 파일 뷰어 기능을 사용하려면 필수입니다. 플레이스토어에서 무료로 설치하실 수 있습니다.

**Q. iOS(아이폰)에서 사용 가능한가요?**  
A. 현재 Android 전용이며, iOS 버전은 추후 출시 예정입니다.

**Q. 보안키를 분실하면 데이터를 복구할 수 있나요?**  
A. 보안키는 재발급이 불가합니다. 반드시 최초 생성 시 안전한 곳에 별도 저장해 주세요.

**Q. Vault 없이 앱만 단독으로 사용 가능한가요?**  
A. 기본 파일 관리 기능은 사용 가능하나, 보안 금고 기능은 Vault 도킹이 필요합니다.

**Q. 네트워크가 없어도 작동하나요?**  
A. 예. D-GO는 기본적으로 폐쇄형(오프라인) 동작이 핵심입니다. 시점인증 서비스 연동 시에만 일시적 네트워크 접근이 필요합니다.
